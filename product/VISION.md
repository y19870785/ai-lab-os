# AI-Lab 产品愿景

## 使命

打造一个长期演进的 AI Operating System，让超哥通过自然语言就能管理整个工作系统。

## 愿景

AI-Lab 不是一个 Agent 框架，不是一个 ChatBot，不是一个 RAG 工具。

AI-Lab 是一个运行在超哥桌面和 NAS 上的 AI 操作系统。

它的核心能力不是"回答问题"，而是：

1. **记住该记住的**：工作记录、决策、知识、经验，不会因为时间流逝而丢失。
2. **知道该做什么**：待办任务、优先级、截止日期，不需要手动梳理。
3. **连接该连接的**：文档、知识、任务、决策之间建立关联，形成可追溯的工作网络。
4. **持续演进**：随着使用时间增长，AI-Lab 对超哥的工作理解越来越深。

## 核心设计原则

1. **数据优先**：所有 AI 输出必须基于真实数据，禁止 LLM 自由编造。
2. **渐进增强**：从单用户单应用开始，逐步扩展，不做过度设计。
3. **本地优先**：核心数据存储本地，敏感信息不出设备。
4. **模型无关**：不绑定任何特定 LLM，随时可切换。
5. **可追溯**：每个决策、每个任务、每段知识都有来源和时间线。

## 首批产品

### CEO Assistant (v0.32.0)

超哥的个人工作总控助手，是 AI-Lab 的第一个真实业务应用。

核心闭环：
- 工作记录 → 结构化存储 → 可检索
- 任务创建 → 状态追踪 → 每日简报
- 决策记录 → 结果追踪 → 经验积累
- 文档导入 → 知识检索 → 引用回答
- 多轮对话 → 上下文记忆 → 连续工作

## 产品路线图

- **v0.32.0**: CEO Assistant MVP — 首次可用
- **v0.33.0**: 企业微信接入 — 移动端使用
- **v0.34.0**: Investment Office 应用 — 投资场景
- **v0.35.0**: 多应用协作 — 跨应用知识共享
- **v0.40.0**: Plugin Marketplace — 生态扩展
